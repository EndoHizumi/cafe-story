# Cafe Story

WebTaleKit で作られた短編ビジュアルノベルゲームです。
小さな喫茶店を舞台に、友人のユキとのひとときを楽しむ作品です。

## ストーリー

とある街の片隅にある喫茶店。
久しぶりに会う友人・ユキと、のんびりとしたひとときを過ごす。
注文する飲み物や会話の選択によって、少しだけ異なる反応が楽しめます。

## 登場人物

| 名前 | 説明 |
|------|------|
| ユキ | 主人公の友人。明るく気さくな性格。 |
| マスター | 喫茶店のオーナー。寡黙だが温かい雰囲気。 |

## 遊び方

### 起動

```bash
npm run play
```

ブラウザが自動で開き、タイトル画面が表示されます。

### 基本操作

| 操作 | 動作 |
|------|------|
| クリック / Enter | テキストを進める |
| Ctrl キー（長押し） | スキップ（テキストを素早く送る） |

### 選択肢

テキストの途中で選択肢が現れます。ボタンをクリックして選んでください。
選んだ内容によってユキのセリフが変わります。

**飲み物の選択（1回目）**

- カフェラテを頼む
- ブレンドコーヒーにする
- 紅茶がいいな
- 今は良いよ

**帰るタイミングの選択（2回目）**

- そろそろ帰ろうか → エンディングへ
- もう少し話そう → 会話が続く

### エンディング

「そろそろ帰ろうか」を選ぶとエンディングに進みます。
エンディング後は「タイトルに戻る」からもう一度遊べます。

## 開発者向け

### セットアップ

```bash
npm install
```

### コマンド一覧

| コマンド | 説明 |
|----------|------|
| `npm run play` | ビルド＋ブラウザ自動起動（開発用） |
| `npm run dev` | ビルド＋開発サーバー起動（自動起動なし） |
| `npm run build` | 本番用ビルド |

### 開発サーバー

`http://localhost:8080` で起動します。

### プロジェクト構成

```
cafe-story/
├── src/
│   ├── index.js          # エントリーポイント
│   ├── scene/            # シナリオファイル (.scene)
│   │   ├── main.scene    # メインシーン（喫茶店）
│   │   └── ending.scene  # エンディングシーン
│   ├── screen/           # 画面テンプレート (.html)
│   │   ├── title.html    # タイトル画面
│   │   └── game.html     # ゲーム画面
│   └── resource/         # 画像・音声素材
│       ├── background/   # 背景画像
│       ├── chara/        # キャラクター画像
│       ├── bgm/          # BGM
│       └── se/           # 効果音
├── engineConfig.json     # エンジン設定
└── webpack.config.js     # ビルド設定
```

### シナリオの編集

`src/scene/*.scene` を編集後、`npm run dev` を実行するとコンパイルされます。
シナリオの文法は [WebTaleKit ドキュメント](https://github.com/your-repo/webTaleKit) を参照してください。

## 技術スタック
